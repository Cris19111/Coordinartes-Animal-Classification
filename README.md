**Coordinartes - Animal Classification**

Proyecto pequeño para visualización/gestión de datos de coordenadas y clasificación de animales.

**Estructura**
- **Proyecto:**: Coordinartes-Animal-Classification
- **Archivos principales:**
  - [index.html](index.html) : Interfaz web para visualizar los datos (mapa / resultados).
  - [map.json](map.json) : Dataset de coordenadas y metadatos (formato JSON).
  - [script.py](script.py) : Script Python para procesamiento, clasificación o preparación de datos.

**Requisitos**
- **Python:**: >= 3.8 (si vas a ejecutar `script.py`).
- Dependencias: ninguna lista explícita presente; si `script.py` requiere paquetes, crea `requirements.txt` y usa `pip install -r requirements.txt`.

**Uso**
- Abrir la interfaz: abre [index.html](index.html) en un navegador moderno (doble clic o `http-server` si hace falta).
- Ejecutar el script Python (opcional):

```bash
python3 script.py
```

Si `script.py` toma argumentos, ejecuta `python3 script.py --help` o inspecciona el contenido del fichero.
